# [Connor Bowling's Personal Portfolio](https://connorbowling.com/) - Skills, Experience, and Projects

Personal portfolio site built to showcase my work as a software engineer. Built with React and TypeScript, with a focus on clean UI and smooth animations.

### Building the Site

The main challenge was balancing a clean design with enough content to be useful. I iterated on the layout a fair amount, particularly around how experience and projects are presented, and used Framer Motion for animations throughout.

### Running Locally

```bash
npm install
npm start
```
